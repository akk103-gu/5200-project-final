This is my DSAN 5200 final project, titled "The Borderline F.P". It's a visual essay tracking linguistic changes in a piece of internet slang across 10 years. Data collection and analysis are performed in two separate notebooks, and everything visual is stored either in the style.css file or index.html. I created many illustrations for this project, which are all stored in img.

I was working on my project in a separate repository through the semester, and merged it here right before submission. I opted not to use GU Domains to host my site, and instead chose to use github pages. You can see the site by checking "deployments".
